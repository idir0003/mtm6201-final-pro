# Plant Palette – Final Web Project (MTM6201)

## Project Overview
Plant Palette is a modern vegetarian and vegan restaurant website built using HTML, CSS, and the Bootstrap framework. The goal of this project was to transform my UX design mockups into a fully responsive and accessible multi-page website.

The website includes three pages:
- Home page
- Menu page
- Reservation page

The design focuses on a clean, natural aesthetic using green tones to represent sustainability and fresh plant-based food.

---

## My Process

I started this project by reviewing my Figma wireframes and layout structure from my UX design course. I planned the website structure first, making sure each page had a clear purpose and consistent navigation.

Then I built the HTML structure using semantic elements such as header, nav, section, and footer. After that, I integrated Bootstrap to help create a responsive layout using the grid system.

Once the structure was complete, I styled the website using CSS. I customized the design to match my concept by using green tones, typography choices (Playfair Display and Montserrat), and custom components like the hero section, menu layout, and reservation form.

I tested the website on different screen sizes to ensure it was responsive and adjusted layouts when needed.

---

## Challenges and Solutions

One of the biggest challenges I faced was making the layout responsive, especially the menu section where images and text needed to stay aligned properly on different screen sizes. I solved this by using Bootstrap’s grid system and adjusting flexbox properties in CSS.

Another challenge was styling the reservation form while keeping it readable and visually balanced. I fixed this by organizing the inputs in a column layout and using consistent spacing.

I also had some difficulty maintaining consistent spacing and alignment across all pages, but I improved this by reviewing each section and standardizing margins and padding.

- Responsive layout for different screen sizes
